## Objective
This repository contains statistical modeling solutions for Problems 1–11 of the **Categorical Analysis** course.  
The analyses focus on:

- Generalized Linear Models (GLMs)
- Poisson Regression
- Negative Binomial Regression
- Model Interpretation and Diagnostics
- Count Data Analysis

The project demonstrates the implementation of categorical data analysis techniques using statistical programming tools and real/simulated datasets.

---

## Topics Covered
1. Introduction to GLMs  
2. Logistic Regression  
3. Poisson Regression  
4. Overdispersion in Count Data  
5. Negative Binomial Regression  
6. Model Comparison Techniques  
7. Residual Analysis  
8. Likelihood Ratio Tests  
9. Goodness-of-Fit Evaluation  
10. Parameter Interpretation  
11. Practical Applications of Categorical Analysis
